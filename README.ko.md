## 💼 주요 프로젝트

### SMARTDent v3 (2019.01 - 2020.06)
**통합 의료 영상 플랫폼** | 주식회사 레이

치과 CT/X-Ray 장비를 위한 **마이크로서비스 아키텍처** 설계. 영상 취득 소프트웨어 및 DICOM 2D/3D 시각화 개발. **ISO 13485** 인증을 위한 검증 문서 직접 작성.

**주요 지표**:
- 압축 기준 **15 MB/s** raw-equivalent 전송량 @100Mbps (이론치 12.5 MB/s 대비)
- 레이턴시 **50% 감소** (50ms → 25ms)
- SRS/SDS 문서 **1,200+ 페이지** 작성

**분산 패턴**:
- **Database Gateway Pattern**: 중앙 집중식 데이터 접근 계층으로 분산 노드 간 일관성 보장
- **Event-Driven Sync**: 네트워크 전체 시스템 상태 자동 동기화 (Eventual Consistency)
- **Service Separation**: C++ 백그라운드 서비스(영상 처리) + .NET 프론트엔드(시각화) 분리 아키텍처

**기술 스택**: C++14, C#, WPF, DICOM, PostgreSQL, DCMTK, PACS, TCP/IP, ISO 13485, IEC 62304

**인증**: CE, FDA 510(k), KFDA, CCC
